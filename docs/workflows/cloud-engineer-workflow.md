# Cloud Engineer Workflow

Policy Forge lets cloud engineers remediate first.

1. A scanner finding appears in the queue.
2. The engineer reviews affected assets and severity.
3. The engineer fixes the issue in the cloud environment.
4. Remediation evidence is attached to the finding.
5. The finding is marked ready for validation.
6. Compliance staff review the evidence asynchronously.

The engineer does not wait for compliance approval before reducing risk.
