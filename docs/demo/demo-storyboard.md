# Demo Storyboard

## Audience

CISOs, AOs, ISSOs, assessors, cloud engineers, and security or compliance leaders.

## Story

1. Start with a critical synthetic finding in `acme-regulated-cloud`.
2. Show that the cloud engineer can remediate first.
3. Attach immutable evidence to the finding.
4. Generate or update the POA&M item.
5. Map the finding to `AC-2`.
6. Have the ISSO validate remediation and record attestation.
7. Show assessor review of evidence and control history.
8. Assemble the authorization package from source records.
9. End on the CISO dashboard with drill-down to the underlying data.

## Message

Policy Forge makes compliance prove security work instead of slowing it down.
